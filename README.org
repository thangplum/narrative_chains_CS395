## Installation
The necessary library for the algorithm are: pip, spacy, pandas, neuralcoref, cython

1. spacy
```
pip install -U spacy
```
2. pandas
```
pip install pandas
```
3. cython
```
pip install cython
```
4. neuralcoref
Follow the installation instruction from (https://github.com/huggingface/neuralcoref)
